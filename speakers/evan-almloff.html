Evan Almloff

Organization
DioxusLabs
Evan Almloff is a software engineer at DioxusLabs. In his current role, he is working on making fullstack UI in Rust easy through ergonomic design and tooling while maintaining type safety throughout the application. He has created several key libraries in the Rust web development ecosystem including Manganis, a library for asset collection and optimization, and sledgehammer-bindgen, the fastest Javascript binding library for Rust. He is also the creator of Kalosm, an open-source framework for controllable language, audio, and image models in Rust.
